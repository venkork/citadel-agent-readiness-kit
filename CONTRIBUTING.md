# Contributing

Thank you for improving Citadel agent documentation. This repository is **The Citadel’s public agent-integration docs only** — no product runtime code.

## What we welcome

- Corrections to public Citadel boundaries (pricing routes, product names, agent handoff URLs)
- Clearer agent integration guidance aligned with `llms.txt` and `agents.txt`
- Typo fixes and heading structure improvements

## What we do not accept

- Generic “SaaS kit” positioning that obscures this is Citadel-specific documentation
- Proprietary prompts, API keys, credentials, or internal workflow exports
- Fake statistics, testimonials, or adoption claims
- Free-audit or match-guarantee framing
- Exposure of private audit logic, n8n, or database internals

## How to propose changes

1. Open an issue using the **Docs improvement** template, or fork and open a pull request.
2. Verify changes against live https://thecitadelapp.com/llms.txt and `/agents.txt` where relevant.
3. Keep diffs focused on public agent copy only.

## Review criteria

Accuracy vs live Citadel public surfaces, safe trust boundaries, and no overclaiming.
